Projeto 2 - Automatização em python.

Este projeto foi desenvolvido inteiramente na linguagem de programação python, no intensivão da Hashtag Treinamentos. Onde tinhamos um problema da demora em conseguir fazer o preenchimento de milhares de dados de uma planilha no formato csv no site de uma empresa (Eles simularam esse site para cadastro dos produtos). Foram utilizadas as seguintes bibliotecas python:

Bibliotecas
Pyautogui - Tem como principal foco a automação de processos repetitivos.
Time - Utilizado para criar as pausas necessárias para não correr o risco de fazer o processo muito rápido e bugar algo.
Pandas - Utilizada para ciência de dados e providencia uma abordagem rápida e flexível com estruturas robustas. Utilizamos para puxar os dados existentes na tabela .csv
